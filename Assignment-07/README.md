# Assignment 07

## Assignment 07: CSV File Handling in Python

### Course Details
- **Course Name:** Python Programming Lab (MCA31PC06)
- **Batch:** B3  
- **Academic Year:** 2025–26 (Semester I)

### Aim
To read data from a CSV file and create a list of product objects using Python.

### Description
This assignment focuses on working with CSV files in Python.  
It demonstrates how to:
- Create a CSV file using Python
- Read data from the CSV file
- Store CSV data into a list for further processing

The assignment helps understand structured data handling using the `csv` module.

### Programs Included
1. Create a CSV file containing product details
2. Read the CSV file
3. Store product data into a list
4. Display product information

### Files Included
- Python source file(s)
- CSV file (`products.csv`)
- Assignment PDF

### Outcome
Students learn how to handle CSV files and work with tabular data in Python applications.
