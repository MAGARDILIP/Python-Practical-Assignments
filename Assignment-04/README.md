## Aim
To understand and implement user-defined functions in Python for solving different computational problems.

## Description
This assignment focuses on the practical use of functions in Python. It covers defining functions, passing arguments, returning values, and using built-in functions to solve real-world problems such as number conversion, mathematical computations, list processing, and modular programming.

The assignment helps in improving code reusability, modularity, and logical thinking using functions.

## Programs Covered
1. Find numbers divisible by another number using functions
2. Convert decimal number to binary, octal, and hexadecimal
3. Find ASCII value of a character
4. Calculate H.C.F and L.C.M of two numbers
5. Find factors of a given number
6. Simple calculator using functions
7. Shuffle a deck of cards and draw 5 cards
8. Display calendar of a given month and year
9. Find sum of natural numbers using recursion
10. Demonstrate built-in functions of the math module
