# Assignment 09

## Assignment 09: File Content Analysis in Python

### Course Details
- **Course Name:** Python Programming Lab (MCA31PC06)
- **Batch:** B3  
- **Academic Year:** 2025–26 (Semester I)

### Aim
To read a text file and calculate the number of lines, words, and characters in the file.

### Description
This assignment focuses on text file analysis using Python.  
It explains how to:
- Read the entire content of a text file
- Count the number of lines
- Count words and characters
- Handle file-related exceptions

Such programs are useful in document processing and data validation tasks.

### Programs Included
1. Read a text file
2. Count number of lines
3. Count number of words
4. Count number of characters

### Files Included
- Python source file(s)
- Sample text file
- Assignment PDF

### Outcome
Students understand file reading techniques and text analysis using Python.
