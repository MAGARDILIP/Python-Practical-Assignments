# Assignment 08

## Assignment 08: Generating CSV Reports Using Dictionaries

### Course Details
- **Course Name:** Python Programming Lab (MCA31PC06)
- **Batch:** B3  
- **Academic Year:** 2025–26 (Semester I)

### Aim
To generate a CSV report from a list of dictionaries using Python.

### Description
This assignment demonstrates how dictionary data can be converted into CSV format.  
It covers:
- Creating a list of dictionaries
- Writing dictionary data into a CSV file
- Using `DictWriter` from the `csv` module
- Displaying generated CSV content

This approach is commonly used in real-world reporting systems.

### Programs Included
1. Create product data using dictionaries
2. Generate a CSV report
3. Display CSV contents

### Files Included
- Python source file(s)
- Generated CSV file (`report.csv`)
- Assignment PDF

### Outcome
Students gain practical knowledge of converting dictionary-based data into structured CSV reports.
