## Aim
To understand and implement the basic concepts of **Object-Oriented Programming (OOP)** in Python, including classes, objects, constructors, instance variables, and methods.

---

## Description
This assignment focuses on introducing Object-Oriented Programming concepts using Python.  
It includes programs that demonstrate how to define classes, create objects, initialize data using constructors, and access object attributes and methods.

These programs help in understanding how real-world entities can be modeled using classes and objects.

---

## List of Programs
1. Create a class with attributes and display their values.
2. Demonstrate the use of constructors (`__init__` method).
3. Create multiple objects of a class and access instance variables.
4. Implement methods inside a class to perform simple operations.
5. Display object details using class methods.

- Constructor (`__init__`)
- Instance Variables
- Methods
- Object Creation
- Data Encapsulation

## Outcome
After completing this assignment, the student will be able to:
- Understand basic OOP concepts in Python
