## Aim
To understand advanced concepts of Python functions including default arguments, variable-length arguments, lambda functions, recursion, and nested functions.

## Description
This assignment focuses on advanced function concepts in Python. It includes programs that demonstrate flexible function usage, functional programming techniques, recursion, dictionary handling, and nested functions to solve complex programming problems efficiently.

The assignment enhances logical thinking and helps in writing clean, reusable, and efficient code.

## Programs Covered
1. Calculate area of a rectangle using default arguments
2. Find minimum, maximum, and average of a list
3. Calculate product using variable-length arguments (*args)
4. Sort a list of tuples using lambda function
5. Calculate factorial using recursion
6. Square elements of a list using functions
7. Convert list of strings to uppercase
8. Check whether a number is prime
9. Print key-value pairs of a dictionary
10. Demonstrate nested functions
