## Description
This assignment covers intermediate and advanced OOP concepts in Python.  
It includes programs that demonstrate inheritance, method overriding, use of constructors and destructors, stack-based validation, and problem-solving using classes.

These programs strengthen understanding of how Python supports OOP principles and real-world problem modeling.

---

## List of Programs

1. Create a `Vehicle` class with `max_speed` and `mileage` attributes and display their values.
2. Write a Python class to find a pair of elements from an array whose sum equals a given target.
3. Create a `Bus` child class that inherits from `Vehicle` and adds maintenance charges.
4. Convert an integer number to a Roman numeral using a Python class.
5. Validate a string of parentheses using stack logic.
6. Create a `Car` class with attributes like make, model, and year and display details.
7. Define a `Student` class and display student information.
8. Use a constructor to initialize a `Person` class and display details.
9. Create a `Book` class and demonstrate the use of a destructor.
10. Demonstrate inheritance and method overriding using `Animal`, `Dog`, and `Cat` classes.
