## Aim
To understand and implement built-in Python modules such as os, sys, math, and datetime for system-level and mathematical operations.

## Description
This assignment focuses on the practical use of Python standard library modules. It includes programs related to file and directory handling, command-line arguments, system paths, mathematical computations, and date-time operations.

The assignment helps in understanding how Python interacts with the operating system and performs advanced computations using built-in modules.

## Programs Covered
1. List files and directories using os module
2. Count command-line arguments using sys module
3. Create, change, and remove directories using os module
4. Display system path and environment variables
5. Calculate area of a circle using math module
6. Compute factorial using math module
7. Generate prime numbers using math.isqrt()
8. Calculate square root and round result
9. Find GCD of two numbers using math.gcd()
10. Display current date and time using datetime module
