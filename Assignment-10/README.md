# Assignment 10

## Assignment 10: File Input and Output Operations in Python

### Course Details
- **Course Name:** Python Programming Lab (MCA31PC06)
- **Batch:** B3  
- **Academic Year:** 2025–26 (Semester I)

### Aim
To perform advanced file handling operations such as reading, writing, appending, and processing text files.

### Description
This assignment covers multiple file handling concepts including:
- Reading an entire file
- Appending text to a file
- Reading files line by line
- Finding longest words
- Writing user input into a file
- Working with CSV files
- Generating reports from data

It gives a complete overview of file handling in Python.

### Programs Included
1. Read and display file contents
2. Append text to a file
3. Read file line by line
4. Find longest word(s)
5. Write alphabet data into a file
6. Create file only if it does not exist
7. Read CSV and create product list
8. Generate CSV report
9. Count lines, words, and characters
10. Write user input to a file

### Files Included
- Python source file(s)
- Text files
- CSV files
- Assignment PDF

### Outcome
Students gain strong understanding of file I/O operations and real-world data handling using Python.
